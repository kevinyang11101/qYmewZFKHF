# 求职招聘系统 python954

## 项目概述

本求职招聘系统基于Python语言开发，整合了前端框架Vue和后端框架Django，致力于打造一个功能全面、易于管理的招聘平台。

## 技术栈

- **开发语言**：Python
- **数据库**：MySQL
- **前端框架**：Vue
- **后端框架**：Django
